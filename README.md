# aliyun-cli-skills

通过自然语言自主操作阿里云 CLI，将用户的自然语言指令转换为阿里云 CLI 命令并执行，便捷操作阿里云资源。

## 📋 功能特性

- **自然语言交互**：用日常中文描述需求，无需记忆复杂的 CLI 命令语法
- **100+ 云产品支持**：ECS、SLS、RDS、OSS、SLB 等主流阿里云产品的增删改查操作
- **智能 API 探索**：自动通过 `--help` 探索正确的 API 和参数格式
- **安全操作**：危险操作前确认，支持 `--dryrun` 模拟执行
- **跨平台支持**：Linux、macOS、Windows

## 前置配置
使用前请先调用此 skill ，自然语言描述：安装和配置 aliyun cli 即可。


## 🎯 使用场景

当你需要：
- 查询 ECS 实例状态、启动/停止实例
- 在 SLS 中写入或查询日志
- 管理 OSS Bucket 和对象
- 查询 RDS 数据库实例信息
- 以及其他阿里云资源操作时...

只需说自然语言即可，例如：
- "查询杭州地域的所有 ECS 实例"
- "在 SLS 项目 my-project 的 logstore 中写入一条日志"
- "从 OSS bucket 下载文件到本地"

实用性远超你的想象！
